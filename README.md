Instagram Story Scraper


Features:
- Command-line tool to download stories for one or many usernames.
- Optional Flask web UI for interactive downloads.
- Organizes downloads as /downloads/{username}/{timestamp}_{shortcode}.{ext}
- Stores metadata (username, timestamp, file type) in the filename and per-item JSON.
